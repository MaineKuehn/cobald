=================
Glossary of Terms
=================


.. glossary::

    Opportunistic Resources
        Any resources available for but not dedicated to a specific task.
        This includes resources which are acquired temporarily, but not owned permanently.
        Strongly put, any resource borrowed for usage outside of its dedicated purpose.
        This includes performing a non-dedicated task instead of idling in the absence of a dedicated task.

    Indistinguishable Resources
        Any resources that are equally suited to fulfill the same tasks.
        It is irrelevant which specific resource serves which task.
        This does not imply strict equality, merely that any differences are inconsequential or negligible.

    Pool
        A collection of resources which are :term:`indistinguishable <indistinguishable resources>`.
